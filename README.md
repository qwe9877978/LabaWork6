## Задача 1: Система для работы с котами

### Условие
Реализовать класс `Cat`, который:
- Принимает имя кота при создании
- Может мяукнуть один раз (выводит `<Имя>: мяу!`)
- Может мяукнуть N раз (выводит `<Имя>: мяу-мяу-...-мяу!`)
- Должен поддерживать подсчёт количества мяуканий через обобщённую обёртку
- Должен работать с коллекциями котов

### Алгоритм решения
1. **Интерфейс `IMeowable`**  
   Создан базовый интерфейс с методом `Meow()`, обеспечивающий единообразную работу со всеми "мяукающими" объектами.

2. **Класс `Cat`**  
   - Хранит имя кота с обязательной валидацией (имя не может быть пустым)
   - Реализует два метода `Meow()`:
     * Без параметров — выводит одиночное мяуканье
     * С параметром `n` — генерирует строку из повторяющихся "мяу", разделённых дефисами
   - Проверяет корректность входных данных (количество мяуканий > 0)

3. **Обобщённая обёртка `MeowCounter<T>`**  
   - Использует шаблоны (обобщения) для подсчёта вызовов метода `Meow()`
   - Хранит внутренний счётчик `CallCount`, инкрементируемый при каждом мяуканье
   - Работает с любым типом, реализующим `IMeowable`

4. **Вспомогательные методы в `MeowHelper`**  
   - `MeowAll()` — вызывает мяуканье для всех котов в коллекции
   - `MeowFiveTimes()` — заставляет кота мяукнуть ровно 5 раз

5. **Работа с пользователем**  
   - Реализовано двухуровневое меню для выбора подзадач
   - Все входные данные проходят строгую валидацию
   - Обработка исключений для некорректного ввода имён и чисел

## Задача 2: Система для работы с дробями

### Условие
Реализовать класс `Fraction`, который:
- Представляет дробь в виде числителя и знаменателя
- Поддерживает арифметические операции (+, -, *, /) с другими дробями и целыми числами
- Реализует сравнение дробей
- Обеспечивает клонирование объектов
- Кэширует вещественное значение дроби с использованием шаблонов

### Алгоритм решения
1. **Базовый класс `Fraction`**  
   - Хранит числитель и знаменатель с нормализацией знака (знак всегда в числителе)
   - Проверяет знаменатель на ноль в конструкторе
   - Реализует арифметические операции через работу с общими знаменателями
   - Переопределяет методы `Equals()` и `GetHashCode()` для корректного сравнения
   - Реализует интерфейс `ICloneable` для создания глубоких копий

2. **Кэширование через шаблоны**  
   - Создан интерфейс `ICachedDouble` с методом `GetCachedValue()`
   - Реализован обобщённый класс `CachedFraction<T>`, который:
     * Наследуется от `Fraction`
     * Вычисляет и сохраняет вещественное значение при создании
     * Работает с любым типом-наследником `Fraction` благодаря ограничению `where T : Fraction`

3. **Арифметические операции**  
   - Сложение/вычитание: приведение к общему знаменателю
   - Умножение: числитель×числитель, знаменатель×знаменатель
   - Деление: умножение на дробь с перевёрнутыми числителем и знаменателем
   - Все операции создают новые объекты, не изменяя исходные

4. **Работа с пользователем**  
   - Четыре подзадачи в меню для демонстрации разных возможностей
   - Валидация всех входных данных (знаменатель ≠ 0, корректные числа)
   - Демонстрация клонирования и сравнения с примерами эквивалентных дробей (1/2 = 2/4)

---
